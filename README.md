## 🚀 Projects

### 🔒 [FairChat](https://github.com/divergenttt/FairChat)
> Private messenger with confidential payments. Built on Fairblock + Arc Protocol.

- End-to-end encrypted messages & file attachments
- Confidential crypto payments inside chat
- Powered by Fairblock's threshold encryption

### 🤖 [CrawlPay](https://github.com/divergenttt/crawlpay)
> Monetize your content from AI crawlers. Pay-per-crawl gateway using x402 protocol.

- Automatic paywall for bots & AI crawlers
- USDC micropayments via Circle
- Built with Next.js 14 + Supabase
